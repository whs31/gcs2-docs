```protobuf
package xproto.pb.telemetry;

enum Vehicle {
  option allow_alias = true;
  
  VEHICLE_UNSPECIFIED = 0;
  VEHICLE_VT30 = 1;
  VEHICLE_VT45 = 2;
  VEHICLE_VT440 = 3;
  VEHICLE_VT550 = 3;
}
```

Перечисление, содержащее тип БПЛА в соответствии с протоколом `ruavp` (см. [[UAV Model]]).

Определяется по косвенным признакам (наличие определенных структур и полей в пассивной телеметрии). Способ определения может меняться и является частью деталей имплементации.


| Значение перечисления | Комментарий           | Примечание                                      |
| --------------------- | --------------------- | ----------------------------------------------- |
| `VEHICLE_UNSPECIFIED` | Тип БПЛА не определен |                                                 |
| `VEHICLE_VT30`        | VT30                  |                                                 |
| `VEHICLE_VT45`        | VT45                  |                                                 |
| `VEHICLE_VT440`       | VT440                 |                                                 |
| `VEHICLE_VT550`       | VT550                 | На данный момент является псевдонимом для VT440 |
