Уникальный идентификатор, служащий для определения уникальности и правильности порядка отправки/получения пакетов.

Идентификатор создается при кодировании пакета и равняется последним 16 битам от значения текущего системного времени в секундах:
```c
uint16_t ruavp_generate_sid() {
  return time(NULL) & 0xFFFF;
}
```
