/* Protocol decode/encode error code */
enum
{
    RUAVP_ERROR_NONE,                   /**< No error */
    RUAVP_ERROR_CRC_MISMATCH,           /**< CRC mismatch */
    RUAVP_ERROR_INVALID_DATA,           /**< Invalid data */
    RUAVP_ERROR_INVALID_SIZE,           /**< Invalid size */
    RUAVP_ERROR_NO_ROUTE,               /**< No route for the packet */
    RUAVP_ERROR_OUT_OF_ORDER_PACKET,    /**< Packet is out of order */
    RUAVP_ERROR_DUP_PACKET,             /**< Duplicated packet */
    RUAVP_ERROR_OWN_PACKET,             /**< Packet destination is the same as source */
    RUAVP_ERROR_INVALID_STRUCTURE_ID,   /**< Invalid structure ID */
    RUAVP_ERROR_UNKNOWN_STRUCTURE,      /**< Unknown structure */
    RUAVP_ERROR_INVALID_ADDRESS,        /**< Invalid address */
    RUAVP_ERROR_TOO_BIG_SIZE,           /**< Packet is too big */
};

##### Out-of-order packet
```c
RUAVP_ERROR_OUT_OF_ORDER_PACKET
```

Выбрасывается при нарушении порядка прихода пакетов (т.е. пакет, сериализованный ранее, приходит позже чем те, что были сериализованы позднее). См. [[Контроль потока данных]].
##### Duplicated packet
```c
RUAVP_ERROR_DUP_PACKET
```

Выбрасывается при появлении дубликата пакета. См. [[Контроль потока данных]].
